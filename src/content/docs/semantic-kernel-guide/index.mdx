<Aside type="tip" title="Agent Framework or Semantic Kernel?">
  Microsoft Agent Framework unifies SK + AutoGen. For new projects, consider [Agent Framework](/microsoft-agent-framework-guide/). Semantic Kernel remains fully supported and has the richer plugin/planner ecosystem.
</Aside>
