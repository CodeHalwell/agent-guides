## What's new in v1.2.9 (April 2026)

- **Standard JSON Schema support** (Zod 4, Valibot, ArkType) — removes validator lock-in.
- **`ReducedValue` type** — custom reducers with separate input/output schemas.
- **`UntrackedValue` type** — transient per-run state that isn't checkpointed.
- **`@langgraphjs/toolkit`** — new package for prebuilt agents like `createReactAgent`.

<Aside type="caution" title="Where did createReactAgent go?">
  `createReactAgent` moved from `@langchain/langgraph/prebuilt` to the new `@langgraphjs/toolkit` package. The old path still warns; migrate when convenient.
</Aside>

## Install, minimal example

```typescript
import { StateGraph, START, END, MemorySaver } from '@langchain/langgraph';
import { BaseMessage } from '@langchain/core/messages';

type State = { messages: BaseMessage[] };

const graph = new StateGraph<State>({ channels: { messages: { reducer: (a, b) => [...a, ...b] } } })
  .addNode('respond', async (state) => ({ messages: [/* ... */] }))
  .addEdge(START, 'respond')
  .addEdge('respond', END)
  .compile({ checkpointer: new MemorySaver() });

const result = await graph.invoke(
  { messages: [] },
  { configurable: { thread_id: 'session-1' } }
);
console.log(result);
```
