## What's new in v0.14.3 (April 2026)

- **Sandbox agents** — persistent isolated workspaces with files, directories, Git repos, snapshots. Backends: local Unix, Docker, Blaxel, E2B, Modal, Runloop, Vercel.
- **WebSocket transport** for OpenAI Responses models — low-latency multi-turn conversations via `responses_websocket_session()`.
- **Sync function tools** now run on worker threads via `asyncio.to_thread()`.

<Aside type="caution" title="Breaking changes from v0.6.x">
  - Requires `openai>=2.0` — v1.x is no longer supported.
  - **Python 3.9 dropped**; minimum is now 3.10.
  - MCP tool failures propagate as exceptions by default — configure `failure_error_function` to restore previous behaviour.
  - `Agent#as_tool()` return type narrowed from `Tool` to `FunctionTool`.
</Aside>

## Install, minimal example

```python
from agents import Agent, Runner

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
)

result = Runner.run_sync(agent, "What is 2 + 2?")
print(result.final_output)
```
