## What's new in v0.1.63 (April 2026)

- **Extended thinking configuration** — control reasoning token budgets.
- **File checkpointing** — save and restore session state.
- **Session rewind** — travel back to any turn in a conversation.
- **Computer Use stability improvements** for long-running VM sessions.

## Install, minimal example

```python
from anthropic import Anthropic
from claude_agent_sdk import Agent

client = Anthropic()

agent = Agent(
    client=client,
    model="claude-3-5-sonnet-latest",
    system="You are a helpful assistant.",
)

response = agent.run("What is 15 + 27?")
print(response.content)
```
