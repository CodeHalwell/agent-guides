## What's new in v1.84.1 (April 2026)

- **Durable execution** — checkpointing and resume for long-running agents.
- **Graph support** — typed state graphs with Pydantic nodes.
- **Evals framework** — LLM-as-judge, regression suites, CI integration.
- **Advanced integrations** — Logfire, OpenTelemetry, observability providers.

## Install, minimal example

```python
from pydantic import BaseModel
from pydantic_ai import Agent

class Answer(BaseModel):
    value: int
    reasoning: str

agent = Agent('openai:gpt-4o', result_type=Answer)

result = agent.run_sync("What is 15 + 27?")
print(result.data)  # Answer(value=42, reasoning='...')
```
