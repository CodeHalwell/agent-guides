## What's new in v0.2.116 (April 2026)

- **Package rename** from `@anthropic-ai/claude-code-sdk` → `@anthropic-ai/claude-agent-sdk`.
- **Computer Use** parity with Python SDK.
- **MCP** client and server support — `createSdkMcpServer(...)` helper.
- **Session management** — `listSessions`, `forkSession`, `deleteSession`, `renameSession`, `tagSession`, `InMemorySessionStore`.
- **Hooks** via `HOOK_EVENTS` object + `hooks:` option on `query()` / `ClaudeSDKClient`.

## Install, minimal example

There is no `Agent` class in `@anthropic-ai/claude-agent-sdk` 0.2.116. The public entry points are `query()` (one-shot) and the lower-level exports listed on the package. Earlier drafts of this page documented a `new Agent({...})` constructor — it does not exist.

```typescript
// One-shot query
import { query } from '@anthropic-ai/claude-agent-sdk';

for await (const message of query({
  prompt: 'What is 15 + 27?',
  options: {
    model: 'claude-3-5-sonnet-latest',
    systemPrompt: 'You are a helpful assistant.',
  },
})) {
  console.log(message);
}
```

For richer session flows use the session helpers (`forkSession`, `listSessions`, `InMemorySessionStore`) or the MCP server factory `createSdkMcpServer`.
