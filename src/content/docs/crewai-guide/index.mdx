## What's new in v1.14.2 (April 2026)

- **Checkpoint forking with lineage tracking** — branch execution from any saved checkpoint.
- **`from_checkpoint` on `Agent.kickoff`** — resume a crew run from a named checkpoint.
- **Checkpoint CLI** — `crewai checkpoint resume|diff|prune`.
- **Template management CLI** — `crewai template` commands.
- **Enriched token tracking** — `reasoning_tokens`, `cache_creation_tokens`.
- **Scoped streaming handlers** — per-run isolation across concurrent crews.

<Aside type="caution" title="Breaking changes since v1.5.0">
  - `CodeInterpreterTool` removed from the public API.
  - `CrewStructuredTool` moved to internal API.
  - `stop` parameter no longer sent to GPT-5 / o-series models.
  - v1.10.0 yanked — use 1.10.1+.
</Aside>

## Install, minimal example

```python
from crewai import Agent, Task, Crew, Process, LLM

llm = LLM(model="openai/gpt-4-turbo")

analyst = Agent(
    role="Research Analyst",
    goal="Analyse topics thoroughly",
    backstory="Expert analyst with 10 years of experience",
    llm=llm,
)

task = Task(
    description="Research AI advancements in 2026",
    expected_output="Concise analysis of key developments",
    agent=analyst,
)

crew = Crew(agents=[analyst], tasks=[task], process=Process.sequential)
result = crew.kickoff()
print(result)
```
