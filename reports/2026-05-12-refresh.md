**Install date:** 2026-05-12

**Source findings (`.routine-envs/check-0512-py/lib/python3.11/site-packages/langgraph/`):**

Minor release. New exports in `langgraph.prebuilt`:

- **`ToolRuntime`** (dataclass): passed to tool functions at execution time; exposes `state`, `context`, `config`, `stream_writer`, `tool_call_id`, `store`, `tools`, `execution_info`, `server_info`. Enables tools to inspect current graph state and write to the event stream mid-execution.
- **`ToolCallTransformer`** (abstract class, `abc.ABCMeta`): base class for interceptors that transform tool call arguments before execution.

`langgraph-checkpoint` bumped to 4.1.0 (minor interface improvements, backward-compatible). `langgraph-prebuilt` bumped to 1.1.0 (ships the new `ToolRuntime`/`ToolCallTransformer`).

All core graph symbols unchanged: `StateGraph`, `END`, `START`, `CompiledStateGraph`, `MemorySaver`, `create_react_agent` (no deprecation warning), `ToolNode`, `ValidationNode`, `InjectedState`, `InjectedStore`, `InMemoryStore`, `Command`, `Send`, `Interrupt`, `interrupt`, `entrypoint`, `task`.
